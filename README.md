# Pretty Logger

The Node.js logger library you were looking for

### Install

`npm install prettylogger`


### How to use

```javascript
var logger = require('prettylogger')(options);

logger('your', 'log', 'comes', 'here'); // it gets the same console.log's arguments
```

### Options

module (String)
color (String)
time (Boolean)
timeFormat (String)
