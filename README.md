# Pretty Logger

Just another Node.js fancy logger library.

### Install

`npm install prettylogger`


### How to use

```javascript
var logger = require('prettylogger')(options);

logger('your', 'log', 'comes', 'here'); // it gets the same console.log's arguments
```

### Options

module (String)

color (String)

time (Boolean)

timeFormat (String)
